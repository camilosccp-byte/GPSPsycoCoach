# GPSPsycoCoach: Player Anxiety & Well-being Tracking in R

## 📌 Project Overview
This project explores a deeply specialized area of sports science: using physiological data to monitor a football player's mental and emotional state. By analyzing HRV (Heart Rate Variability / VFR) and HR (Heart Rate / FC) data, this R-driven tool evaluates anxiety levels and mood patterns in players.

## 📊 Methodology & Technical Stack
- **Language:** R
- **Data Inputs:** Heart Rate Variability (VFR) and Heart Rate (FC) time-series data.
- **Objective:** Tracking physiological stress markers to predict psychological fatigue and emotional burnout.

## 🚀 Key Insights for Coaching Staffs
1. **Mental Health Monitoring:** Provides a quantifiable metric to understand a player's anxiety and emotional well-being.
2. **Preventive Therapy:** Helps medical and coaching staffs apply timely psychological or conditioning therapies before performance drops.
3. **Optimized Recovery:** Complements standard GPS physical data with emotional workload tracking.
